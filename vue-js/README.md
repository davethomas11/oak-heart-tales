# vue-js

## Build game from python ( Transcrypts game to javascript and places it in the vue project folder )
- Requires venv with transcrypt loaded
```bash
sh transcrypt.sh
```


## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm run serve
```

### Compiles and minifies for production
```
pnpm run build
```

### Lints and fixes files
```
pnpm run lint
```
